# Ultrasonic Radar System 

# Overview
An Arduino-based radar system designed to detect the 
position, distance and speed of objects in real time. 
The system uses an ultrasonic sensor mounted on a servo 
motor which rotates and sends ultrasonic waves — 
reflected waves are analyzed and displayed visually 
on screen.

# Components Used
- Arduino UNO
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Breadboard & Connecting Wires

# Software Used
- Arduino IDE
- Processing Software

# How It Works
1. Servo motor rotates the ultrasonic sensor
2. Sensor sends ultrasonic waves in different directions
3. Waves reflect back after striking an object
4. Arduino analyzes the reflected waves
5. Output displayed on screen showing distance 
   and position of object

# Key Features
- Detects objects within 2 to 40 cm range
- Real-time object position and distance display
- Low power consumption
- Open source Arduino code
